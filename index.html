Creativity & Connection

Ever since I was little I wanted to connect to people through my creativity. I would stand up in front of my elementary classes and read out epic stories I had written, many including my classmates. In high school I moved on to art and painted murals all around the building. In college I learned how to engage people the best way through graphic design and art exhibitions. And now I love the endless possibilities of constructing great user experience in web development and programming. It's like this huge puzzle that has a huge reach across the world!

Skills

Javascript | React / Redux | HTML / CSS | Python | Flask | Express | Node.js | Git | SQL / SQLAlchemy / Sequelize | Alembic | Jinja | Pug

I graduated from AppAcademy, completing their 6-month full-time intensive Full-Stack Software Engineering program. This was a fast-paced environment and coupled with my already challenge-seeking character, I gained the tools to quickly learn emerging languages, programs, and APIs by myself. I have extensive experience building applications using React/Redux with a variety of backend frameworks including Flask and Express connected to custom built database schemas using PostgresSQL. I built custom user authentication using Bcrypt and also created fully functioning apps with templating engines such as Jinja and Pug. I'm especially proud of my React.js and Flask Midnight Oil project where I cloned Brainscape/Anki to make a history themed flashcard creation and studying app that is intuitive to use and dynamically updates decks and cards to reflect a user's current proficiency.

Dev Projects

Full-Stack projects built by me!

Midnight Oil

A clone of Brainscape and Anki based off the phrase "burning the midnight oil", meaning to study late into the night. It is a flashcard making and studying application with the theme of late Renaissance battle adventures as you progress your mastery level of the decks/cards you make.

  • Live
  • Github

Crypt Seeker

Clone of HipCamp, but for those interested in staying in supernatural places. There are some good ole fashioned Haunted Hotels, Cryptid hideaways, mysterious spaces, and otherwise supernatural places!

  • Live
  • Github

Petsy

Fullstack clone of Etsy, an online marketplace where people sell, buy, and collect unique pet items. Collaborated with: Andrew Murray, Lucas Monteiro, Yanelys Mena.

  • Live
  • Github
